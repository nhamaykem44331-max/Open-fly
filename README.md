# OpenFly

OpenFly is a B2C flight booking PWA for the Vietnam market.

This repository is a Turborepo monorepo with:
- `apps/api` — NestJS API skeleton
- `apps/web` — Vite + React + TypeScript PWA skeleton
- `apps/admin` — Next.js admin panel skeleton

Detailed implementation plan: `../KEHOACH-OPENFLY-BACKEND.md`.

## Quick Start

```bash
npm install
npm run dev
```

Run type checks before committing:

```bash
npm run type-check
```
